# GNOME DeBloater (GDB)

Debloater for supported distros on GNOME

Prerequisites:
- curl (for ubuntu/debian: ``sudo apt install curl -y``)

```
cd; wget https://raw.githubusercontent.com/agustux/GNOME-DeBloater-GDB-/main/gdb.sh && bash gdb.sh
```

| Distros Supported | Bloated RAM Usage | Debloated RAM Usage|
|------------------|-------------------|--------------------|
| Ubuntu 26.04 | 1250 MB used/1700 MB buff/cache |   -> 1050 MB used/900 MB buff/cache |
| Ubuntu 24.04 | 1150 MB used/1700 MB buff/cache |    -> |
| Fedora 43 | 1600 MB used/1850 MB buff/cache |  -> |
| Debian 13 | 1000 MB used/600 MB buff/cache |   ->| 
| Manjaro | 1300 MB used/1700 buff/cache | -> |
